What is colorkik?

Colorkik is a colorful website. This site generate color palettes using images clicked by photographers. Search for 'nature', because mother nature is the most beautiful after all.

Where images come from?

Well there are a lots of site that provide open source images. This site also use an open source api, provided by Pexels, to fetch images.

How color palettes are generated?

Color palettes are generated by using an open source library, Color Cube. This is an open source library which is also created by Vikram

How to make request any changes?

Well, you can fork the project at Github. And make your own changes. You can tweet about the changes you want to make.
